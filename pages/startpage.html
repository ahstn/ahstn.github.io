Start Page

A modern way to start your web browsing

Overview

A minimal, Material Design inspired browser home/new tab page designed to act as a personal dashboard created using AngularJS.

Featuring night and day themes, which rotate based on time by default, but can be user controlled. User customisable bookmarks and links, which can be used as quick access to websites they frequent. A built in RSS reader, which allows users to display their favourite RSS feeds on panels. All of which can be controlled from a custom settings panel, which saves to HTML5 browser local storage.

In the future I hope to convert the project into a broswer extension, thus making it easier to use for potential users.

For more infomation, you can view the git repo hosted online by GitHub
